About the platform

Coding conventions are a set of coding guidelines used by software developers to improve the readability of source code, increase software maintainability, and promote the reuse of coding patterns. Understanding coding conventions has become an indispensable discipline in software engineering. We developed CCBase, a knowledge base of coding conventions, that was constructed from online resources. CCBase contains 3139 coding conventions for Java and C++, with 3758 entities and 767 relations. Furthermore, we have extended the usability of CCBase by developing a question answering system on the base--CocoQa and evaluated it on a coding convention QA dataset. The results show that CocoQa can answer questions about coding conventions precisely. In particular, CocoQa can achieve a precision of 82.92% and a recall of 91.10%.
