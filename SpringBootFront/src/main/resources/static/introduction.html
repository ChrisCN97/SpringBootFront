About the platform

Coding conventions are a set of coding guidelines used by software developers to improve the readability of source code, increase software maintainability, and promote the reuse of coding patterns. Understanding coding conventions has become an indispensable discipline in software engineering. We developed CCBase, a knowledge base of coding conventions, that was constructed from online resources. CCBase contains 3139 coding conventions for Java and C++, with 3758 entities and 767 relations. Furthermore, we have extended the usability of CCBase by developing a question answering system on the base. We have conducted experiments to evaluate CCBase. The experimental results show that CCBase has a wide coverage on entities and relations in coding conventions domain, and the QA system achieves an F1 score of 84.5% on 214 questions raised in StackOverflow.
